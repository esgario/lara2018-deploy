ML-SERVER

Servidor para execução dos modelos de machine learning desenvolvidos usando Flask-Gunicorn-Redis-Pytorch. Toda a comunicação é realizada via API

ENDPOINTS:

/api/inference (Método: POST) - Envia os dados para que o modelo possa realizar a inferência. Retorna um json com o {job_id}.

/api/status/{job_id} (Método: GET) - Consulta o progresso do worker.

/api/result/{job_id} (Método: GET) - Retornar os resultados da inferência.
